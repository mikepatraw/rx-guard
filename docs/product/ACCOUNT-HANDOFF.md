# Prompt Opinion Account Handoff

You do **not** need a Prompt Opinion account for local RXsignal development.

You **do** need a Prompt Opinion account for the following tasks:
- creating the actual RXsignal agent in-platform
- inspecting required configuration fields
- publishing to the Prompt Opinion Marketplace
- validating in-platform invocation
- recording the final compliant demo video

## Current status

RXsignal has been configured/published as a Prompt Opinion BYO/A2A agent, and local/staging repository work can continue without account access. Account access is still needed for final in-platform validation, marketplace URL capture, and the submission video.
